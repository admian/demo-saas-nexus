# Nexus Analytics — SaaS Landing Page

A modern, dark-themed SaaS landing page demo built with **React**, **TypeScript**, **Vite**, and **Tailwind CSS**.

![SaaS Landing Page](https://img.shields.io/badge/React-19-61DAFB?logo=react)
![TypeScript](https://img.shields.io/badge/TypeScript-5-3178C6?logo=typescript)
![Tailwind CSS](https://img.shields.io/badge/Tailwind-4-06B6D4?logo=tailwindcss)

## Live Demo

> Deploy to Vercel and paste your URL here:
> `https://your-demo.vercel.app`

## Features

- Responsive design (mobile, tablet, desktop)
- Hero with dashboard mockup
- Features grid, stats bar, pricing cards
- Email CTA section
- Smooth scroll navigation

## Tech Stack

- React 19 + TypeScript
- Vite 7
- Tailwind CSS 4

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

```bash
npm run build
npm run preview
```
